Developer disk cleaner

Cleanium finds the stuff that eats your disk: node_modules, venvs, Cargo build dirs, Docker unused images and volumes - and lets you clear it out. Windows only.

Download for Windows

Free · Open source · GitHub

See it in action

What it does

  • Scan - Node, Python, Rust/Cargo, Go, Docker, Next/Nuxt, build dirs, package caches.
  • Filter - Limit the list by category (e.g. only Docker or only Node).
  • Delete - File-system items go to Recycle Bin; Docker items (dangling images, stopped containers, unused volumes, build cache) removed via Docker API.
  • History - SQLite stores past scans; reopen any run to review findings.
  • Export - Save the current result set as JSON.

Use with care

Deleting is real. File-system deletes go to the Recycle Bin so you can restore until you empty it. Docker removals are permanent. The app skips system and IDE dirs, but always check the path before hitting Delete.
